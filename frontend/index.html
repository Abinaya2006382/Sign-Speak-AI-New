Technology Framework Architecture

Sign Speak AI is an end-to-end full-stack computer vision application powered by local machine learning processing pipelines.

Python Flask Backend

Flask acts as the core REST server coordinating DB interactions, processing raw images, running MediaPipe solutions, and feeding landmark arrays to TensorFlow classifiers.

MediaPipe Tracking

Extracts hand models from the input video frames in real-time, mapping 21 landmark nodes containing coordinates in 3-dimensional Cartesian coordinates.

TensorFlow Keras

A multi-layer feed-forward deep neural network (DNN) trained on normalized structural landmarks, predicting probabilities across gestures via Softmax.

SQLite Database

Provides persistent local storage for custom vocabulary lists (gesture names and joint structures) and detailed log metrics of translation history.

MediaPipe Hand Landmarks Mapping

Landmark Reference Key

Every detected frame produces 21 coordinates (indexed 0 to 20). The spatial relationships, relative lengths, and angles are parsed into a normalized 1D vector:

  • 0: Wrist
  • 1 - 4: Thumb (CMC, MCP, IP, Tip)
  • 5 - 8: Index Finger (MCP, PIP, DIP, Tip)
  • 9 - 12: Middle Finger (MCP, PIP, DIP, Tip)
  • 13 - 16: Ring Finger (MCP, PIP, DIP, Tip)
  • 17 - 20: Pinky (MCP, PIP, DIP, Tip)
Calibration Note: The system is calibrated based on hands facing the camera at an angle where all knuckles are clearly visible. Extremely slanted hands or overlapping fingers may skew the prediction confidence scores.
